# 🎮 Hero Stats System

A simple web-based hero database that lets users search heroes and view their stats, counters, strong matchups, and team compatibility.

This project is still **under development** and will be expanded over time.

---

## 🌐 Live Website

👉 https://bloxdropgg.github.io/hero-stats-site/

---

## ⚙️ Current Features

- 🔎 Hero search system
- 🧠 View hero profiles
- ⚔️ Counters (who beats who)
- 🤝 Compatibility (best teammates)
- 🖼 Hero images support
- 📁 JSON-based data system

## 📁 How It Works

- All hero data is stored in `heroes.json`
- The website dynamically loads data using JavaScript
- Adding or editing heroes in the JSON file updates the site automatically
